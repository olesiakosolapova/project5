## Простой сайт с использованием ASP.NET MVC и Entity Framework

### Магазин смарт часов

Сайт состоит из трех страниц: Главная, Магазин (на которой можно посмотреть товар) 
и Админ (на которой можно Добавить, Изменить и Удалить товар).
